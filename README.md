# B33th0v3n
Scripts to sort lots of music file by several criterias 👂🎶📂

## 🚀 How to use it

* Install requirements
```sh
pip3 -r requirements.txt
```
* Download datasets
```sh
python3 app.py -g 
``` 

* Check help guidelines
```sh
python3 app.py -h 
```
